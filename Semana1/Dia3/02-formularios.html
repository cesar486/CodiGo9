Elementos de los formularios

Número Nombre del control Descripción
1 input
  • Es un campo de entrada de diferentes tipos de datos
  • es el elemento mas usado en los formularios
  • Tipos de Input
    • text sirve ingresar texto plano.
    • email sirve para ingresar texto con formato de correo electronico
    • number sirve para ingresar numeros enteros y decimales
    • date sirve para seleccionar de un picker, una fecha determinada
    • time sirve para ingresar horas en formato am y pm
    • password sirve para ingresar contraseñas
    • color sirve para seleccionar un color de un paleta de colores
    • file sirve para seleccionar un archivo local de multiples formatos
    • reset es un control en forma de boton que limpia o borra todos los campos del formulario
    • checkbox control para seleccionar uno más elementos de un grupo.
    • radio control para seleccionar un único elemento de un grupo.
2 textarea
  • Es un control para ingresar texto plano en mas de una fila
3 select
  • Es un cuadro de opciones desplegable
  • Se puede elegir un elemento de la lista de opciones
4 button
  • Es un control que representa un botón
5 label
  • Representa una etiqueta de texto
  • Por lo general, acompaña a cualquier elemento de ingreso de datos
6 fieldset
  • Sirve para agrupar controles dentro de un formulario
7 legend
  • Representa el titulo de un elemento
  • por lo general, acompaña al fieldset
